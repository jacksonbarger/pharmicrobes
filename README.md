# 🧫 PharMicrobes — Med Micro Study Hub

Interactive medical microbiology study app covering Lectures 1 & 2.

## Features
- 📖 **Study Guide** — Complete breakdown of all lecture content
- ⭐ **Ultra High-Yield** — Must-memorize facts with expandable details
- 🗂️ **Bug Charts** — Filled + blank Gram stain classification
- 🃏 **Flashcard Quiz** — 34 organism cards with progress tracking
- 🌳 **Bug ID Tree** — Click-through diagnostic decision tree
- 🧠 **Mnemonics** — 9 custom memory aids
- ⚡ **Confusables** — Side-by-side comparison cards

## Deploy
Push to GitHub and connect to Vercel, or:
```bash
npm i && npm run build && npm start
```

## Stack
Next.js 15 + Tailwind CSS 4 + React 19
