# Content Drop Folder

Drop lecture materials here (`.docx`, `.pdf`, `.pptx`, `.md`, `.txt`, images, CSVs).

Claude reads `../CLAUDE.md` to know how to turn these into study tools. See §2 and §5 there.

## Suggested structure

```
content/
├── lecture-03-antifungals/
│   ├── slides.pdf
│   ├── notes.md
│   └── worksheet.docx
└── lecture-04-antivirals/
    └── slides.pptx
```

Organizing by lecture is optional but makes it easier for Claude to know which source a given fact came from.
